# GKE & Monitoring Stack (Grafana & Prometheus)

## Stack :
Ubuntu 24

## Preparation

1. Installing Requirement
```
sudo apt-get install ca-certificates gnupg curl
```

2. add gcloud repository
```
echo "deb [signed-by=/usr/share/keyrings/cloud.google.gpg] https://packages.cloud.google.com/apt cloud-sdk main" | sudo tee -a /etc/apt/sources.list.d/google-cloud-sdk.list
```

3. gcloud installation
```
gcloud init
```

When prompted to sign in, accept and then sign in to your Google Account in your browser. Click Allow to grant permission to access resources.

## GKE Installation

**Note :** First, we need to enable kubernetes engine on project that we use.

4. Install kubectl
```
sudo apt-get install kubectl
```

5. Install gcloud auth plugin
```
sudo   apt-get install google-cloud-sdk-gke-gcloud-auth-plugin
```

6. create installer *.sh 
```
vi install-gke.sh
```
Fill with this syntax, set the variable requirement based on resource (especially on PROJECT_ID) :
```sh
#!/bin/bash

# Set your project ID
PROJECT_ID="your-project-id"
# Set your preferred region or zone
REGION="asia-southeast2"
ZONE="asia-southeast2-a"
# Set your cluster name
CLUSTER_NAME="gke-playground"

# Configure gcloud to use your project
gcloud config set project $PROJECT_ID

# Create the GKE cluster
gcloud container clusters create $CLUSTER_NAME \
    --zone=$ZONE \
    --num-nodes=2 \
    --machine-type=e2-standard-8 \
    --disk-type=pd-standard \
    --disk-size=100GB \
    --enable-autoscaling \
    --min-nodes=2 \
    --max-nodes=4 \
    --scopes=https://www.googleapis.com/auth/cloud-platform \
    --addons=HttpLoadBalancing,GcpFilestoreCsiDriver

# Verify cluster creation
gcloud container clusters list

# Get credentials to interact with the cluster
gcloud container clusters get-credentials $CLUSTER_NAME --zone=$ZONE

# Verify nodes
kubectl get nodes
```

7. give permission
```
chmod +x install-gke.sh
```

8. Run script 
```
./install-gke.sh
```

## Monitoring Stack Installation
9. add repository
```
helm repo add prometheus-community https://prometheus-community.github.io/helm-charts/
```
```
helm repo add stable https://charts.helm.sh/stable/
```

10. Update helm repo
```
helm repo update
```

11. Start installation
```
helm install prometheus prometheus-community/kube-prometheus-stack
```

12. Get password of grafana 
```
kubectl get secret --namespace monitoring grafana -o jsonpath="{.data.admin-password}" | base64 --decode ; echo
```

13. portforwarding from localhost to pod
```
kubectl port-forward deployment/prometheus-grafana 3000
```
