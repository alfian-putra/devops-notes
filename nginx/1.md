Configuring Nginx :
```sh
sudo touch /etc/nginx/sites-available/load-balancer
sudo ln -s /etc/nginx/sites-available/load-balancer /etc/nginx/sites-enabled/load-balancer
sudo vi /etc/nginx/sites-available/load-balancer
```

fill this :
```nginx
upstream backend {
    server localhost:8001;
    server localhost:8002;
    server localhost:8003;
}

server {
    listen 80;

    location / {
        proxy_pass http://backend;
    }
}
```

from the server we using port 80 redirect anything from 0.0.0.0:80/ to backend which containing server that we run previously.

```sh
systemctl restart nginx
```

**Note :** dont forget to remove default configuration from site-available and it symlink on site-enabled.
